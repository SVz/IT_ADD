        IT Section includer by SV 
       ---------------------------
       
This tool is intended to insert a new section from file into exe.
It useful when you have a new import table file generated by an
import rebuilder tool and you want to insert it in your unpacked exe. 

How it works:

First click on 'Load Exe' to choose your unpacked exe.
Click on 'IT File' to open the import table file.
'Do it' show save dialog windows, and save a new exe.

This tools doesn't modify import table offset in PE Header directory.
You have to do it !
